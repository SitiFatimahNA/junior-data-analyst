# Data Lake

## Pendahuluan

Tidak semua data memiliki bentuk tabel seperti pada database SQL. Saat ini perusahaan juga menyimpan gambar, video, file JSON, PDF, hingga log aplikasi. Untuk mengelola berbagai jenis data tersebut digunakan Data Lake.

---

## Apa itu Data Lake?

Data Lake adalah tempat penyimpanan yang mampu menyimpan data mentah (raw data) dalam berbagai format tanpa harus mengubah strukturnya terlebih dahulu.

...

## Kesimpulan

Data Lake memberikan fleksibilitas dalam menyimpan data dan menjadi fondasi penting dalam Big Data serta Machine Learning.
