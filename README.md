# Junior Data Analyst

Repository ini berisi dokumentasi pembelajaran, roadmap, artikel, catatan materi, dan proyek selama mengikuti **Program Simulasi Industri Data Analyst**. Repository ini disusun sebagai portofolio sekaligus personal branding untuk menunjukkan proses belajar dan hasil pekerjaan selama program berlangsung.

---

## Tujuan Repository
- Mendokumentasikan proses belajar Data Analyst secara terstruktur
- Menyimpan catatan materi, latihan, dan tugas
- Membangun portofolio yang dapat diverifikasi
- Menunjukkan progres pembelajaran selama simulasi industri

---

## Roadmap Program
Program ini mengikuti alur pembelajaran simulasi industri Data Analyst, meliputi:
1. Orientasi Industri & Setup Tools
2. Data Collection & Understanding
3. Data Cleaning & Preparation
4. Exploratory Data Analysis
5. Problem Solving & Business Question
6. Reporting & Data Storytelling
7. Final Project & Portofolio
8. Presentasi Akhir & Evaluasi
