# Junior Data Analyst

Repository ini berisi dokumentasi proses pembelajaran, artikel, dan perkembangan materi selama mengikuti **Program Simulasi Industri Data Analyst**. Repository ini dibuat sebagai tempat menyimpan hasil belajar, catatan perkembangan, serta portofolio yang dapat menunjukkan perjalanan belajar saya di bidang data.

---

## Tujuan Repository
Repository ini dibuat untuk:

- mendokumentasikan proses belajar Data Analyst secara bertahap,
- menyimpan artikel, catatan, dan hasil pembelajaran selama program berlangsung,
- membangun portofolio pembelajaran yang dapat dilihat dan diverifikasi,
- menunjukkan progres belajar selama mengikuti simulasi industri.

---

## Tentang Repository Ini
Repository ini akan terus diperbarui mengikuti materi, tugas, dan aktivitas yang dikerjakan selama program simulasi industri berlangsung. Struktur dan isi repository dapat bertambah atau berubah menyesuaikan kebutuhan pembelajaran, dokumentasi, serta pengembangan portofolio.

---

## Konten yang Saat Ini Tersedia

### 1. Roadmap / Artikel Pembelajaran
Folder `roadmap/` berisi artikel dan dokumentasi pembelajaran awal terkait bidang data, di antaranya:

- **01-perbedaan-role-data.md**  
  Membahas perbedaan antara **Data Analyst**, **Data Scientist**, dan **Data Engineer**.

- **02-peta-belajar-data-untuk-pemula.md**  
  Membahas peta belajar data untuk pemula, mulai dari dasar data, Excel, SQL, Python, visualisasi data, hingga arah pengembangan karier.

---

## Rencana Pengembangan Repository
Ke depannya repository ini akan diisi secara bertahap sesuai proses belajar dan tugas selama program, seperti:

- dokumentasi materi yang dipelajari,
- catatan pembelajaran tools dan konsep data,
- latihan atau studi kasus,
- hasil analisis data,
- serta proyek atau portofolio yang dikerjakan selama simulasi industri.

---

## Penutup
Repository ini merupakan bagian dari perjalanan belajar saya untuk memahami dunia data, khususnya pada jalur **Junior Data Analyst**. Melalui repository ini, saya ingin mendokumentasikan proses belajar secara terstruktur, sekaligus membangun portofolio yang dapat menunjukkan perkembangan kemampuan saya dari waktu ke waktu.
